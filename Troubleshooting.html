Singleplayer lobby shows no players connected

This is usually due to a misconfigured firewall. Durak requires ports to be forwarded on the computer in order to function. In windows, these ports can be configured as follows:

Windows 10

  1. Open the search menu
  2. Enter "Allow an app through Windows Firewall", and select the result
  3. Press the "Change settings" button at the top, you may be prompted for Administrative rights
  4. After allowing changes to be made, press the "Allow another app..." button at the bottom of the screen
  5. If Durak is not listed, press "Browse", otherwise select it from the list and continue
    • Locate "DurakGame.exe" on your machine
  6. Press the "Add" button

Windows 8/8.1

  1. Open the search menu (swipe left or Windows key + Q to bring up charms bar)
  2. Enter "Control Panel" in the search panel and hit enter
  3. Select "Windows Firewall"
  4. Select "Allow a program or feature through Windows Firewall" from the left hand side
  5. Press the "Change settings" button at the top, you may be prompted for Administrative rights
  6. After allowing changes to be made, press the "Allow another app..." button at the bottom of the screen
  7. If Durak is not listed, press "Browse", otherwise select it from the list and continue
    • Locate "DurakGame.exe" on your machine
  8. Press the "Add" button

Windows 7

  1. Open the start menu
  2. Open the Control Panel
  3. Select "Windows Firewall"
  4. Select "Allow a program or feature through Windows Firewall" from the left hand side
  5. Press the "Change settings" button at the top, you may be prompted for Administrative rights
  6. After allowing changes to be made, press the "Allow another app..." button at the bottom of the screen
  7. If Durak is not listed, press "Browse", otherwise select it from the list and continue
    • Locate "DurakGame.exe" on your machine
  8. Press the "Add" button

Players cannot connect from outside my network

To connect to a remote server via Direct Connect, a few steps must be taken to properly set up the server:

  • The host's ports need to be forwarded
  • The client needs to know the host's Public IP Address
This requires configuring both the host's machine and the host's router. First off the host needs to allow the game through the local firewall using the following steps:

Windows 10

  1. Open the search menu
  2. Enter "Allow an app through Windows Firewall", and select the result
  3. Press the "Change settings" button at the top, you may be prompted for Administrative rights
  4. After allowing changes to be made, press the "Allow another app..." button at the bottom of the screen

  6. After allowing changes to be made, press the "Allow another app..." button at the bottom of the screen

Afterwards, the host will need to port forward port 55440 on their router. This process differs for different router manufacturers and models. We recommend following the tutorials on PortForward.com for your specific router model.

Program Crashes on Attempting to Make or join a Game

The most common cause of this bug is an incorrect network setup. The most common way to fix this is to disable all unused network adapters.
In Windows, you can use the following steps to view and disable your adapters:

Windows 10

  1. Open the search menu
  2. Enter "View Network Connections", and select the control panel app
  3. Right click the adapter to disable and select "Disable"

Windows 8/8.1

  1. Open the search menu (swipe left or Windows key + Q to bring up charms bar)
  2. Enter "Control Panel" in the search panel and hit enter
  3. Select "View Network Connections" from the left hand side
  4. Right click the adapter to disable and select "Disable"

Windows 7

  1. Open the start menu
  2. Open the Control Panel
  3. Select "Network and Sharing Center"
  4. Select "View Network Connections" from the left hand side
  5. Right click the adapter to disable and select "Disable"

Program Crashes when running Multiple Servers

Running multiple servers on one machine is currently not supported by the Durak game engine. We hope to release a fix for this in the future.

Reporting a Bug

If you find a bug in the application, you can make an issue on our Github page. To do this, you will need to create a Github account. Once you have a Github account, you can post on our Issues page.
